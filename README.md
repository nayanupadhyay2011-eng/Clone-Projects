## ⚙️ How to Run Locally

Follow these simple steps to explore the code or run the projects on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```

2. **Navigate to the project directory:**
   ```bash
   cd YOUR-REPO-NAME
   ```

3. **Launch a project:**
   * Open the desired project folder.
   * Open the `index.html` file directly in any modern web browser, or use the VS Code **Live Server** extension for real-time changes.
